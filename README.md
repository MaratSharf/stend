## Виртуальная среда

### Создание виртуальной среды
```bash
python -m venv venv
```

### Активация виртуальной среды

**Windows:**
```bash
venv\Scripts\activate
```

**Linux/macOS:**
```bash
source venv/bin/activate
```

### Установка зависимостей
```bash
pip install -r mes_production/requirements.txt
```

### Деактивация виртуальной среды
```bash
deactivate
```
